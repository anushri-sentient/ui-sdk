2 Your First Component

Create your first component using either HTML tags or JavaScript API:

<!-- Simple Card Component --> <ui-card title="Welcome to UI Components" content="This is your first component!" actionText="Get Started" actionLink="#" ></ui-card>
// Create a card using JavaScript API const card = UI.card({ title: "Welcome to UI Components", content: "This is your first component!", actionText: "Get Started", actionLink: "#" }); // Insert it into the page UI.insert('#first-demo-js', card);
3 Multiple Components

Combine different components to create rich interfaces:

<!-- Product Card with Badge --> <ui-card title="Premium Product" subtitle="Luxury Category" image="https://picsum.photos/400/250" content="High-quality product with amazing features." actionText="Buy Now" badge="Featured" ></ui-card> <!-- User Table --> <ui-table headers="Name,Email,Role,Status" rows="John Doe,john@example.com,Developer,Active;Jane Smith,jane@example.com,Designer,Active" striped="true" ></ui-table> <!-- Success Alert --> <ui-alert type="success" message="Product added to cart successfully!" duration="3000" ></ui-alert>
// Create multiple components programmatically const productCard = UI.card({ title: "Premium Product", subtitle: "Luxury Category", image: "https://picsum.photos/400/250", content: "High-quality product with amazing features.", actionText: "Buy Now", badge: "Featured" }); const userData = [ { name: "John Doe", email: "john@example.com", role: "Developer", status: "Active" }, { name: "Jane Smith", email: "jane@example.com", role: "Designer", status: "Active" } ]; const userTable = UI.table(userData, { striped: true }); const successAlert = UI.success("Product added to cart successfully!"); // Insert all components UI.insert('#multiple-demo-js', productCard + userTable + successAlert);
