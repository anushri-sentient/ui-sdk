📖 How to Use Design Variants

Add the designVariant attribute to any component to apply these styles

HTML Usage

<ui-card title="My Card" content="Card content" designVariant="glassmorphism"> </ui-card>

JavaScript API

UI.card({ title: "My Card", content: "Card content", designVariant: "glassmorphism" });

Visual Builder

Use the Component Builder and select your preferred design variant from the dropdown menu.
