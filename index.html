Card

Versatile content containers with images, titles, descriptions, and action buttons. Perfect for product displays, articles, and feature highlights.

ðŸ“‹ Available Attributes

  • title - Main heading text
  • subtitle - Secondary heading text
  • content - Main content description
  • image - Image URL for the card
  • actionText - Button text
  • designVariant - Style variant (glassmorphism, neumorphism, etc.)
  • size - Card size (small, medium, large)
<ui-card title="Product Name" subtitle="Product Category" content="This is a beautiful product description that showcases the item." image="https://example.com/image.jpg" actionText="Learn More" designVariant="glassmorphism" size="large"> </ui-card>
Use designVariant to apply different visual styles. Available variants: glassmorphism, neumorphism-soft, claymorphism-clay, brutalism-sharp, aurora-vibrant, minimal-ultra, cyberpunk-neon

Badge

Small status indicators and labels for highlighting important information, notifications, or categories.

ðŸ“‹ Available Attributes

  • label - Text content of the badge
  • color - Color theme (primary, success, warning, danger, info)
  • designVariant - Style variant for different visual effects

Perfect for status indicators, notifications, and category labels. Use color to convey meaning (success=green, warning=yellow, danger=red).

Avatar

User profile images and icons with different sizes and fallback options. Supports images, initials, and emoji.

ðŸ“‹ Available Attributes

  • src - Image URL or emoji
  • alt - Alt text for accessibility
  • size - Size variant (small, medium, large)
  • name - User name for initials fallback
<ui-avatar src="https://example.com/user.jpg" alt="User Profile" size="large" name="John Doe"> </ui-avatar>
If no image is provided, the component will show initials from the name attribute. Supports emoji as src for quick icons.

ðŸ“‹ Available Attributes

  • items - Comma-separated list of chip labels
  • color - Color theme (primary, success, warning, danger)

Great for filtering interfaces, tag systems, and multi-select components. Removable chips emit events when clicked.

Table

Data tables with sorting, filtering, and responsive design. Perfect for displaying structured data, user lists, and reports.

ðŸ“‹ Available Attributes

  • headers - Comma-separated column headers
  • rows - Semicolon-separated rows with comma-separated cells
  • striped - Alternating row colors (true/false)
  • hoverable - Row hover effects (true/false)
<ui-table headers="Name,Email,Role,Status" rows="John Doe,john@example.com,Developer,Active;Jane Smith,jane@example.com,Designer,Active;Bob Johnson,bob@example.com,Manager,Inactive" striped="true" hoverable="true"> </ui-table>
Use semicolons to separate rows and commas to separate cells within rows. Enable striped and hoverable for better readability.

Accordion

Collapsible content sections for organizing information. Perfect for FAQs, settings panels, and content that needs to be expandable.

ðŸ“‹ Available Attributes

  • items - JSON array of accordion sections
  • multiple - Allow multiple sections open (true/false)
<ui-accordion items='[ {"title":"What is this library?","content":"A modern UI component library built with web standards."}, {"title":"How do I get started?","content":"Check the documentation and try the visual builder."} ]' multiple="false"> </ui-accordion>
Use for FAQs, settings, and any content that benefits from progressive disclosure. Set multiple="true" to allow multiple sections open.

Carousel

Image carousel with auto-play functionality and navigation controls. Perfect for showcasing products, portfolios, or image galleries.

ðŸ“‹ Available Attributes

  • images - Comma-separated image URLs
  • interval - Auto-play interval in milliseconds
<ui-carousel images="https://picsum.photos/800/400?random=1,https://picsum.photos/800/400?random=2,https://picsum.photos/800/400?random=3" interval="5000"> </ui-carousel>
Set interval to 0 to disable auto-play. Images should be the same dimensions for best results. Supports touch/swipe gestures.

Alert

Notification messages for user feedback, warnings, and important information. Auto-dismissible with different severity levels.

ðŸ“‹ Available Attributes

  • type - Alert type (success, info, warning, danger)
  • title - Alert title/heading
  • message - Alert message content
  • dismissible - Auto-dismiss after duration (true/false)
  • duration - Auto-dismiss duration in milliseconds
<ui-alert type="success" title="Success!" message="Your changes have been saved successfully." dismissible="true" duration="5000"> </ui-alert>
Use different types to convey meaning: success (green), info (blue), warning (yellow), danger (red). Set duration for auto-dismiss.
