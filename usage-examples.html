Card Component

Create beautiful cards with images, content, and actions. Perfect for product displays, blog posts, and content showcases.

<ui-card title="Product Name" subtitle="Product Category" image="https://picsum.photos/400/250" content="This is a beautiful product description that showcases the features and benefits." actionText="Buy Now" actionLink="#" badge="New" variant="elevated" size="medium" ></ui-card>
// Simple card with defaults const card = UI.card({ title: "Product Name", content: "Product description here" }); // Advanced card with all options const advancedCard = UI.card({ title: "Premium Product", subtitle: "Luxury Category", image: "https://picsum.photos/400/250", content: "Detailed product description with features and benefits.", actionText: "Add to Cart", actionLink: "/cart", badge: "Featured", variant: "elevated", size: "large", rating: "4.8", features: ["Free Shipping", "30-Day Return", "24/7 Support"] }); // Insert into page UI.insert('#card-demo-js', card);

Use the Visual Component Builder to create cards with a user-friendly interface.

Table Component

Display data in organized tables with sorting, filtering, and responsive design.

<ui-table headers="Name,Email,Role,Status" rows="John Doe,john@example.com,Developer,Active;Jane Smith,jane@example.com,Designer,Active" striped="true" hoverable="true" ></ui-table>
// Create table from data array const userData = [ { name: "John Doe", email: "john@example.com", role: "Developer", status: "Active" }, { name: "Jane Smith", email: "jane@example.com", role: "Designer", status: "Active" }, { name: "Bob Johnson", email: "bob@example.com", role: "Manager", status: "Inactive" } ]; const table = UI.table(userData, { striped: true, hoverable: true, sortable: true }); UI.insert('#table-demo-js', table);
// Auto-generate table from any data structure const salesData = [ { product: "Widget A", sales: 150, revenue: "$1,500" }, { product: "Widget B", sales: 200, revenue: "$2,000" }, { product: "Widget C", sales: 75, revenue: "$750" } ]; const salesTable = UI.table(salesData, { striped: true, bordered: true }); UI.insert('#table-demo-data', salesTable);

Alert Component

Show notifications, warnings, and feedback messages to users with auto-dismiss functionality.

<ui-alert type="success" message="Operation completed successfully!" duration="5000" dismissible="true" ></ui-alert>
// Create custom alert const alert = UI.alert("This is a custom message", "info", { duration: 3000, dismissible: true }); UI.insert('#alert-demo-js', alert);
// Quick alert methods UI.success("Success message here"); UI.error("Error message here"); UI.warning("Warning message here"); UI.info("Info message here");

Badge Component

Display status indicators, labels, and notifications with different colors and sizes.

<ui-badge label="New" color="primary" size="medium"></ui-badge> <ui-badge label="Active" color="success" size="small"></ui-badge> <ui-badge label="Warning" color="warning" size="large"></ui-badge>
// Create badges programmatically const badges = [ UI.badge("New", "primary"), UI.badge("Active", "success"), UI.badge("Warning", "warning"), UI.badge("Error", "danger") ].join(''); UI.insert('#badge-demo-js', badges);

Accordion Component

Create collapsible content sections for FAQs, documentation, and organized information.

<ui-accordion items='[{"title":"Section 1","content":"Content for section 1"},{"title":"Section 2","content":"Content for section 2"}]' multiple="false" defaultOpen="0" ></ui-accordion>
// Create accordion from array const items = [ { title: "How to use components?", content: "Simply include the component files and use the custom HTML tags." }, { title: "Can I customize styles?", content: "Yes! All components use CSS custom properties for easy theming." }, { title: "Is it responsive?", content: "Absolutely! All components are fully responsive and mobile-friendly." } ]; const accordion = UI.accordion(items, { multiple: true, defaultOpen: 0 }); UI.insert('#accordion-demo-js', accordion);
